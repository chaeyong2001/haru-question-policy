서비스 소개 수집하는 정보 로컬 저장 권한 데이터 삭제 English Summary

1. 서비스 소개

하루질문: 소복일기는 사용자가 하루를 돌아보며 질문에 답하고, 감정과 배경, 스티커를 함께 기록할 수 있는 라이프스타일/다이어리 앱입니다.

이 앱은 의료 또는 상담 서비스가 아니며, 개인 기록을 위한 도구입니다. 심리치료, 상담, 진단, 위기 대응 서비스를 제공하지 않습니다.

2. 수집하는 정보

현재 버전에서는 사용자의 기록 데이터를 외부 서버로 전송하지 않습니다. 사용자가 작성한 답변, 선택한 감정, 배경, 스티커, 기록, 알림 설정, 언어/글씨체/테마 설정, 앱 잠금 관련 정보는 기기 내부에 로컬 저장됩니다.

3. 수집하지 않는 정보

현재 버전에서는 앱 운영자가 다음 정보를 서버로 수집하지 않습니다.

  • 이름, 이메일, 전화번호 등 계정 정보
  • 위치 정보
  • 연락처
  • 결제 정보
  • 광고 식별자 기반 분석 정보
  • 사용자의 다이어리 내용을 외부 서버로 전송하는 기능

4. 로컬 저장 데이터

다이어리 답변, 감정, 질문 기록, 배경/스티커 선택, 알림 설정, 앱 테마, 글씨체, 언어 설정은 사용자의 기기 내 로컬 저장소에 저장됩니다. 앱을 삭제하거나 기기 설정에서 앱 데이터를 초기화하면 해당 데이터가 삭제될 수 있습니다.

5. 알림 권한

앱은 사용자가 설정한 시간에 오늘의 질문 알림을 표시하기 위해 알림 권한을 요청할 수 있습니다. 알림은 앱 설정에서 켜거나 끌 수 있습니다.

6. 이미지 저장 기능

앱은 사용자가 기록 카드를 이미지로 저장하는 기능을 제공합니다. Android 버전과 저장 방식에 따라 사진/미디어 저장 관련 권한 또는 시스템 저장 동작이 사용될 수 있습니다. 이 기능은 사용자가 직접 저장을 선택했을 때 사용됩니다.

7. 앱 잠금/PIN

앱 잠금/PIN 기능은 개인 기록을 보호하기 위한 로컬 보호 기능입니다. PIN 관련 정보는 기기 내에 저장됩니다. 이 기능은 편의적 보호 수단이며, 고급 보안 또는 완전한 암호화를 보장하지 않습니다.

8. 데이터 삭제

사용자는 앱 설정의 데이터 관리 기능을 통해 저장된 다이어리 기록을 삭제할 수 있습니다. 앱을 삭제하거나 기기 설정에서 앱 데이터를 초기화하는 경우에도 로컬 데이터가 삭제될 수 있습니다.

9. 제3자 제공 여부

현재 버전에서는 사용자의 다이어리 기록을 외부 서버로 전송하거나 제3자에게 제공하지 않습니다.

10. 광고 및 분석 도구

현재 버전에는 광고 SDK, 분석 SDK, 결제 SDK가 포함되어 있지 않습니다. 추후 광고, 분석, 결제, 백업 또는 클라우드 기능이 추가되는 경우 개인정보처리방침이 업데이트될 수 있습니다.

11. 아동 개인정보

이 앱은 아동을 대상으로 설계된 서비스가 아닙니다. 법정 보호자의 동의가 필요한 연령의 사용자는 보호자와 함께 앱 사용 여부를 결정해야 합니다.

12. 개인정보처리방침 변경

앱 기능, 권한, 광고/분석/결제/백업 기능 추가, 관련 법령 또는 정책 변경에 따라 본 개인정보처리방침이 변경될 수 있습니다. 중요한 변경이 있을 경우 앱 또는 정책 페이지를 통해 안내할 수 있습니다.

15. English Summary

Dear Me: Little Questions is a lifestyle diary app for personal reflection. It is not a medical, counseling, therapy, diagnosis, or crisis support service.

  • The current version has no account login and no server sync.
  • Diary answers, moods, backgrounds, stickers, and settings are stored locally on the user’s device.
  • The current version does not transmit diary records to an external server.
  • Notification permission may be used for daily question reminders.
  • Image/media saving behavior may be used when the user saves a diary card image.
  • App lock/PIN is a local convenience protection feature and does not guarantee advanced security or full encryption.
  • The current version does not include advertising, analytics, payment SDKs, or cloud backup.
  • If ads, analytics, payments, backup, or cloud features are added later, this policy may be updated.
